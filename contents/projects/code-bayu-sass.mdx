You can use all css utilities from this package like Bootstrap or Tailwind.

## NPM

This awesome package already published on npmjs. Check [here](https://www.npmjs.com/package/@codebayu/sass)

## Package Installation

Install the package as a dependency by running the following command:

```bash
# npm
npm i @codebayu/sass

# yarn
yarn add @codebayu/sass
```

Import the CSS file on the root directory

```jsx
import '@codebayu/sass/css/index.css'
```

And yeah, Happy coding!
